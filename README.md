# Fraz Anwer - Engineering Manager Portfolio

This is a premium, modern personal CV website designed to position **Fraz Anwer** as a high-impact Engineering Manager. It is built with vanilla HTML, CSS, and JS, requiring no build steps or dependencies.

## 📁 Project Structure
- `index.html`: Core structure and SEO-optimized content.
- `styles.css`: Modern executive design system using Slate & Indigo.
- `script.js`: Interactive animations and navigation logic.
- `assets/`: (Optional) Put images and profile photos here.
