# Fraz Anwer - Engineering Manager Portfolio

This is a premium, modern personal CV website designed to position **Fraz Anwer** as a high-impact Engineering Manager. It is built with vanilla HTML, CSS, and JS, requiring no build steps or dependencies.

## 📁 Project Structure
- `index.html`: Core structure and SEO-optimized content.
- `styles.css`: Modern executive design system using Slate & Indigo.
- `script.js`: Interactive animations and navigation logic.
- `assets/`: (Optional) Put images and profile photos here.

---

## 🚀 Deployment Instructions

This project is built for **Free & Instant Deployment**.

### 1. GitHub Pages (Recommended)
1.  **Create a Repo**: Create a new repository on GitHub named `about-fraz-anwer` (or similar).
2.  **Upload Files**: Push all files in this folder to the `main` branch.
3.  **Enable Pages**:
    - Go to **Settings** > **Pages**.
    - Under **Build and deployment**, select **Deploy from a branch**.
    - Select **main** branch and `/ (root)` folder.
    - Click **Save**.
4.  **Live**: Your site will be live at `https://yourusername.github.io/repo-name`.

### 2. Netlify (Drag & Drop)
1.  Go to [Netlify Drop](https://app.netlify.com/drop).
2.  **Drag & Drop**: Drag the entire `about-fraz-anwer` folder onto the page.
3.  **Live**: Your site is live immediately! You can later configure a custom domain or change the site name in settings.

### 3. Vercel
1.  Install Vercel CLI (optional) or use the dashboard.
2.  **Dashboard**:
    - Go to [Vercel Dashboard](https://vercel.com/new).
    - Import your GitHub repository.
    - Keep settings at default (no build command needed).
    - Click **Deploy**.
3.  **Live**: Your site is live on a `.vercel.app` domain.

## 🛠️ Personalization
To make the site truly yours:
- **Profile Photo**: Replace the SVG/Gradient placeholder in the "About" section with an `<img>` tag pointing to your photo.
- **Resume**: Ensure `Fraz_Anwer_CV_Latest-1.docx.pdf` is in the root directory. If you update the file, make sure the name matches in `index.html`.
- **Metrics**: Update the numbers in the "Experience" and "About" sections to reflect your latest achievements.
